hexo搭建个人博客

发表于2021-09-20|更新于2021-10-16|折腾日记
|阅读量:

想自己搭建博客，但又不想买服务器，亦或是不满意第三方的博客平台。因而可以利用hexo博客框架+github仓库 搭建免费的博客。

W3C介绍及搭建

准备工作

manjaro环境下搭建

需要下列程序

  • Node.js
  • Git （manjaro下自带）

先安装npm

n模块是用来管理nodejs的

安装nodejs v12

安装hexo

#选择一个安装目录，在该目录下执行下面的指令

修改一个配置文件node_modules/hexo-cli/lib/console/init.js把第12行改为const GIT_REPO_URL = 'https://hub.fastgit.org/hexojs/hexo-starter.git';

在hexo目录下初始化一个工作目录” blog “，作为博客根目录

进入该目录

执行下面的命名后才能用：hexo s、hexo g、hexo d这三个命令

将node_modules/.bin添加到局部环境变量中

更改主题

主题推荐：

  • butterfly&安装文档

在博客根目录下

安装pug 以及 stylus 的渲染器

修改根目录下的_config.yml中的theme为butterfly

主题具体配置可以参考butterfly官网

插件安装

博客内页面搜索插件

将博客部署到github

建立github仓库

在github新建一个仓库，命名为XXX.github.io #XXX为你的github名，格式固定

使用ssh连接github

  • 先生成一个密钥
  • 在github设置里面的SSH and GPG keys中添加一个新的密钥，然后将生成的公钥复制到里面
  • 配置git的登录用户
  • 测试是否成功

修改 网站根目录下的_config.yml，在文件的最后面 deploy处

安装hexo-deployer-git工具

将博客部署到github上

测试

浏览器访问 xxx.github.io #xxx为你的用户名

错误合集

报错
