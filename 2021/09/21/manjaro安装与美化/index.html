安装majaro

前期准备

  • 下载manjaro KDE镜像 官网链接

  • 用rufus软件刻录启动盘下载链接

  • 我选择的是低版本的 rufus-3.0p.exe 便携版本(可以选择GPT+UEF模式)

怎么进入安装界面不多介绍，可自行百度

分区方案

我的电脑配置：16G内存、512G固态

目录 大小 文件系统 标记 挂载点
boot 512M FAT32 boot boot/efi
swap 16G linuxswap swap
/ 128G ext4 /
/home 剩下的全部 ext4 /home

因为装的软件的配置文件一般都保存在/home目录下，所以home目录的空间耗费的快很多，需要把大部分的空间给/home目录；之前我给根目录/200G,发现根本用不完。

路径：系统设置=>外观

  • Plasma样式：默认的Breath2 Dark
  • 窗口装饰元素：Breezemite 并将按钮大小设置为巨大
